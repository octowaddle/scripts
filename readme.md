# Scripts

A collection of useful scripts to automate processes.

## Firefox Nightly

### Install Firefox Nightly

```sh
sudo bash -c "$(curl -sLo- https://git.io/JGv2h)"
```

### Remove Firefox Nightly

```sh
sudo bash -c "$(curl -sLo- https://git.io/JGv2h) --remove"
```
